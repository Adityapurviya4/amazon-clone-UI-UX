## Amazon Clone UI/UX 🛒

This project is a front-end Amazon clone built to replicate the core UI/UX design of the Amazon e-commerce platform. The goal of this project is to practice modern web design, responsive layouts, and user interface development.

## ✨ Features
• Amazon-like homepage layout
• Responsive navigation bar with search functionality
• Product listing sections
• Shopping cart UI
• Footer with multiple sections similar to Amazon
• Fully responsive design for desktop, tablet, and mobile

## 🛠️ Technologies Used
• HTML5
• CSS3

## 🎯 Purpose
This project was created for learning and practicing front-end development by recreating the UI/UX of a popular e-commerce website.

## 📌 Note
This is a practice project and is not affiliated with Amazon.
